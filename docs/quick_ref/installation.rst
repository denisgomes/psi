Installation
============

Install from PyPi_
------------------

To install PSI, type the following command in the terminal:

.. code:: sh

    $ pip install psi --user

.. note::
    Python 3.5 or above must already be installed on your system. If you are
    on windows, pip should be available after the install.

    If pip was not installed by default, get and install it using the
    directions provided `here <https://pip.pypa.io/en/stable/installing/>`_.


Install From `Source`__
-----------------------

If you're reading this from the README, i.e. you have the source distribution,
you can install PSI using:

.. code:: sh

    $ python setup.py install --user
